# Decryption-of-Yo-kai-Watch-Elder-and-YSP
This is a Python programme derived from konosuFear’s version, ported to Python 3. It adds support for decrypting Yo-kai Watch YSP update files, in addition to the Elder files, as well as the option to extract sound files or not. Do not enable the entry extraction option for version 3 of the YSP. This script will be modified in the future.
